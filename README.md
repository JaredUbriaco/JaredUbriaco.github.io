# jaredubriaco.github.io

Personal GitHub Pages website.

## Deploy to GitHub Pages

1. Create a new repository on GitHub named **exactly** `jaredubriaco.github.io`
2. Initialize git and push this project:

   ```bash
   git init
   git add .
   git commit -m "Initial commit - Hello World"
   git branch -M main
   git remote add origin https://github.com/jaredubriaco/jaredubriaco.github.io.git
   git push -u origin main
   ```

3. On GitHub, go to **Settings â†’ Pages** and ensure the source is set to "Deploy from a branch"
4. Your site will be live at **https://jaredubriaco.github.io** within a few minutes
